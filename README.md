# homebrew-tap

Homebrew formulae for [serpro69](https://github.com/serpro69) projects.

## Install

```bash
brew install serpro69/tap/capy
```

## Formulae

| Formula | Description |
|---------|-------------|
| [capy](https://github.com/serpro69/capy) | Context-aware MCP server for LLM context reduction |

## How it works

The `capy` formula is automatically updated by the [release workflow](https://github.com/serpro69/capy/blob/master/.github/workflows/release.yml) on each stable release. Do not edit `Formula/capy.rb` manually.
